# Belle-Ecommerce-Site

[![Python](https://img.shields.io/badge/Python-3.11-blue?logo=python&logoColor=white)](https://www.python.org/)
[![Django](https://img.shields.io/badge/Django-4.2-green?logo=django&logoColor=white)](https://www.djangoproject.com/)

**Belle-Ecommerce-Site** is a modern and fully functional e-commerce platform built using **Django** and **Python**. This website enables users to explore and purchase a wide variety of products including **men's and women's clothing, shoes, jewelry, cosmetics, and accessories**. The project demonstrates a **professional, production-ready workflow** including backend management, frontend templating, static/media handling, and database integration.

---

## Project Overview
Belle-Ecommerce-Site is designed to provide an **intuitive, seamless shopping experience** for users. Customers can browse products, view detailed pages with multiple images, and explore categories easily.  

Key Highlights:
- End-to-end web development using Django's **MTV architecture**  
- Fully responsive templates for desktop and mobile  
- Product categories designed for real-world e-commerce scenarios  
- Admin panel for managing products, images, and categories  
- SQLite database for fast development and testing (easily switchable to MySQL/PostgreSQL)

This project is ideal for **portfolio demonstration**, showcasing skills in **backend development, frontend templating, and full-stack integration**.

---

## Features & Functionality
### User Features
- Browse products by category and subcategory  
- View detailed product pages with images, descriptions, and pricing  
- Filter products by category, price, and type  
- Responsive design ensures a smooth experience on mobile devices  

### Admin Features
- Add, update, or delete products easily  
- Upload and manage product images  
- Categorize products for better browsing  
- Full control over the content without touching code  

### Technical Features
- Built with **Python 3** and **Django 4.x**  
- Frontend uses **HTML5, CSS3, and Bootstrap**  
- Supports **static and media file management**  
- Database powered by **SQLite** (development-ready, scalable to production)  
- Fully modular architecture for maintainability and scalability

## Product Categories
- **Men's Clothing** – Shirts, T-shirts, pants, jackets  
- **Women's Clothing** – Dresses, tops, skirts, jackets  
- **Shoes** – Casual, formal, sports, and sandals  
- **Jewelry** – Rings, necklaces, earrings, bracelets  
- **Cosmetics** – Skincare, makeup, and accessories  
- **Accessories** – Bags, watches, belts, sunglasses  

Each category contains multiple products with **detailed pages**, images, and pricing for a **realistic online shopping experience**.

## User Experience Flow
1. **Homepage:** Users see featured products and categories.  
2. **Category View:** Click a category to explore all products.  
3. **Product Detail Page:** View images, description, price, and availability.  
4. **Admin Panel:** Admin can manage products, images, and categories easily.  
5. **Responsive Design:** Website works seamlessly on **desktop, tablet, and mobile devices**.
